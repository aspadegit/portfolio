#3: WAD File Reader & Writer

Photo by Goran Ivos on Unsplash

About

This was made with C++. The purpose of this project was to learn about file systems via reading and editing .WAD files, which is the file type that the video game DOOM is in.

This project involved reading in the bytes of the WAD file and storing appropriate variables, then shifting bytes as necessary when writing to the WAD after adding files. Files could be added via a daemon that could traverse the directories in the WAD file as if it was a normal Linux directory.

#4: 3D Audio Video Game

Photo by Kiran CK on Unsplash

About

This was made in C# with Unity. The purpose of this project was to learn about 3D audio, user studies, and HRTFs (Head Related Transfer Functions).

This project allows the user to traverse a 3D space, and the goal is to locate birds using 3D audio.
