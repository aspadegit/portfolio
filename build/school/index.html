#2: P2P Networking Service

Photo by Chris Ried on Unsplash

About

This was a group project that was made with Java socket programming, and made heavy use of threading.

This project is a peer-to-peer networking service, where several peers can communicate in a network to receive a single file. Individual peers communicate which pieces of the file they have, and exchange those pieces with their fastest neighbors. More details can be found on the GitHub's readme.

#3: WAD File Reader & Writer

Photo by Goran Ivos on Unsplash

About

This was made with C++. The purpose of this project was to learn about file systems and memory management through reading and editing WAD files. WAD is the file extension that the video game DOOM is stored in.

To read WAD files, the bytes of the file were traversed and stored in appropriate variables. To edit WAD files, both the header bytes and lump data had to be written to, and bytes had to be shifted so that the WAD file maintained its proper structure. Files could be added via a daemon that could traverse the directories in the WAD file using the standard Linux command line functions.

#4: 3D Audio Video Game

Photo by Kiran CK on Unsplash

About

This was made in C# with Unity. The purpose of this project was to learn about 3D audio, user studies, and HRTFs (Head Related Transfer Functions).

This project allows the user to traverse a 3D space. The goal is to locate birds as fast as possible using audio cues.
